# Application de Suivi de Jeûne

Application web Flask pour suivre vos jours de jeûne.

## Fonctionnalités
- Définir le nombre de jours de jeûne
- Marquer les jours complétés
- Vue calendrier
- Barre de progression

## Installation locale
```bash
pip install -r requirements.txt
python app.py
```

## Déploiement
Cette application peut être déployée sur Render, Heroku, ou PythonAnywhere.
